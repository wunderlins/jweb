Java Web Technologies Demo

  • Servlet - a simple get/post example using servlet and a HTML template.

    This example uses the servlet net.wunderlin.jweb.servlet.Servlet class as well as /META-INF/templates/form.html.

  • JSP - a simple get/post example using JSP.

    This example uses a single jsp file called /jsp.jsp. The jsp. is mapped to the url /JSP.html through web.xml for explanatory reasons (which is optional, it could also be called by it's real file name).

  • JSTL - JSP Standard Tag Library

    This example is implemented in a single file called /jstl.jsp and is mapped trough web.xml to the URI JSTL.html.

    JSTL is part Java EE 5, more on the topic in the following references:

    1. Standardizing JSP, Part 1 , OnJava, August 2002
    2. JSP Standard Tag Libraries, Part 2, OnJava, May 2002
    3. JSP Standard Tag Libraries, Part 1, OnJava, March 2002
